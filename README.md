# 📈 TRIM: Trajectory Analysis in Minutes

TRIM is a lightweight, Streamlit-based web application for fast trajectory analysis of single-cell RNA-seq data using Diffusion Pseudotime (DPT).

## Features
- **Fast Upload**: Support for `.h5ad` and `.csv` files.
- **Trajectory Calculation**: Diffusion Pseudotime (DPT) with root cell selection.
- **Premium Visualization**: Interactive UMAP, gene dynamics heatmaps, and boxplots.
- **Export**: Download results as `.h5ad` or `.csv`.

## Quick Start
Run `run_trim.bat` on Windows to launch the application.

## Requirements
- Python 3.9+
- Scanpy
- Streamlit
- Plotly
- Matplotlib
- Pandas
